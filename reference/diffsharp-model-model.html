this.reverseDiff ?tag

Full Usage: this.reverseDiff ?tag

Parameters:
    ?tag : uint32 - The level tag for nested differentiation. Defaults to the current global nesting level

Adjust the parameters of the model to include support for reverse-mode automatic differentiation.

After this call the current parameters of the model will support reverse-mode propagation. After the completion of the corresponding reverse operation, the computed derivative will be available.

?tag : uint32

The level tag for nested differentiation. Defaults to the current global nesting level
